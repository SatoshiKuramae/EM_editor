操作説明
Add GameObject->オブジェクトの追加
Export JSON->Jsonファイルに書き出し
Import JSON->Jsonファイル読み込み
Derete Object->	選択中のオブジェクトの削除
Pattern n/n->パターン変更

リスト内の「オブジェクトｎ」をクリックするとパラメータを操作できるようになります

概要欄
Selected Objectに選択中のオブジェクトナンバーが記載されてます。が、わかりずらいのでもっとわかりやすくしてる途中
Z,Cボタンで視点を動かせます（注視点は原点）
Patternボタンでウェーブごとの敵配置が行えます。
Patternを帰るたびにインポートが行われますが、後から手動でインポートする形になるかも
セーブしない限りインポートを行えば最後に保存された状態までで戻ります。リセットとしても使っても良いが注意せよ
