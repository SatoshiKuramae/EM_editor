操作説明
Add GameObject->オブジェクトの追加
Export JSON->Jsonファイルに書き出し
Import JSON->Jsonファイル読み込み


オブジェクトを生成するとリストにObeject～のように表示される
リスト上に表示されているObjectをクリックするとパラメータ操作が行える



概要蘭
・Import Jsonは2回行わないでください
一度インポートしたら終了前にエクスポートしてください。データが無くなります
